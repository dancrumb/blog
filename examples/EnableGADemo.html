Enable Google Analytics Demo

This page demonstrates EnableGA.js. This is a script which searches for external links on a webpage and enables Google Analytics tracking of these links. See this post for more details.

In this demonstration, there are a list of links below. Some are internal, some are external. When you click on the external ones, you will be shown the pseudo-locations that will be tracked by Google Analytics.
